How to Measure a Furnace Filter Correctly

Best for: you keep buying the wrong filter size or it fits loosely/tightly.

Key idea: Filters often have a “nominal” size (rounded) and an “actual” size (slightly smaller).

Tools

  • Tape measure
  • Phone camera (optional)

Step-by-step

  1. Turn the system off at the thermostat before removing the filter.
  2. Remove the old filter and note airflow arrow direction.
  3. Measure length and width across the frame (in inches).
  4. Measure thickness (commonly 1", 2", 4", or 5").
  5. Write down the printed size and your measured size.
  6. Buy the same nominal size as printed when possible.
  7. Install new filter with arrow pointing toward the blower/furnace.
